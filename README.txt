food
====
This project is an analysis of spending patterns on food in Los Angeles and San Francisco. The file main.py contains all the code 
that was used in the analysis while the file Food_Analysis.pdf contains the results of the analysis along with an interepretation
of some of the results. 

The following Python libraries are required for to run the file main.py:
1. Numpy
2. Matplotlib
3. Pandas
4. Patsy
5. Statsmodels
